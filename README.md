# Информационная система бюджетирования и финансового контроля

## Описание

Информационная система бюджетирования и финансового контроля - это программа для автоматизации процессов планирования, учета и анализа финансовых показателей предприятия. Система позволяет работать с бюджетами доходов и расходов (БДР), движением денежных средств (БДДС), центрами финансовой ответственности (ЦФО), а также проводить план-факт анализ и сценарное моделирование.

## Цель проекта

- Автоматизировать процессы финансового планирования и контроля
- Обеспечить учет плановых и фактических показателей по центрам ответственности
- Реализовать контроль лимитов расходов и анализ отклонений
- Предоставить инструменты сценарного моделирования для прогнозирования

## Структура

- **auth/** - модули для авторизации и регистрации пользователей
- **database/** - модуль подключения к базе данных PostgreSQL
- **modules/** - модули для основных окон приложения:
  - Ввод план/факт показателей
  - Управление центрами ответственности (ЦФО)
  - Управление статьями бюджета
  - Формирование БДР и БДДС
  - План-факт анализ
  - Контроль лимитов
  - Сценарное моделирование
  - Модуль для формирования и экспорта отчёто
- **main.py** - файл для запуска проекта

## Как установить и запустить?

- Нажать в репозитории на Code и после на download zip
- Распаковать ZIP-файл проекта
- Установить необходимые библиотеки: `pip install PyQt5 psycopg2`
- Указать в консоли путь в папку: `cd (путь)`
- Настроить параметры подключения к БД в файле `database/db.py`
- Запустить проект через команду: `python main.py`
- Войти под учетной записью администратора (логин: `admin`, пароль: `admin`)

## Важно

- Нужно иметь установленный Python (версия 3.7 или выше)
- Необходим PostgreSQL сервер для хранения данных
- Требуется установить библиотеки: **PyQt5**, **psycopg2**, **hashlib**
- При первом запуске автоматически создаются все необходимые таблицы в БД
- Создаются тестовые данные: центры ответственности, статьи бюджета, сценарии
- Система поддерживает две роли пользователей: **admin** (полный доступ) и **user** (только просмотр)
